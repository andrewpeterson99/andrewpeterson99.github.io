Instructions

This program generates sine waves at random frequencies and for random durations within user-specified bounds.
Specify minimum and maximum frequencies within the top two boxes (click to type in values).
Specify minimum and maximum durations within the bottom two boxes.
Please don't type in anything that's not a number. I didn't code an error-catcher for this...
